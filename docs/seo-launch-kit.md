## Part 1 — Google Search Console (≈5 minutes, do first)

Google has no public API to request indexing of normal pages, so this is manual.

1. Go to https://search.google.com/search-console (you already verified the domain).
2. Confirm the sitemap is submitted: **Sitemaps** → enter `sitemap-index.xml` → Submit.
   (It may already be there from before — just confirm "Success".)
3. For each NEW url below: paste it into the **URL Inspection** bar at the top →
   wait for the check → click **Request Indexing**. Do all of them:
   - https://lovefarfox.com/blog/how-to-make-a-long-distance-relationship-work
   - https://lovefarfox.com/blog/long-distance-relationship-statistics
   - https://lovefarfox.com/blog/long-distance-date-ideas
   - https://lovefarfox.com/blog/long-distance-relationship-texts
   - https://lovefarfox.com/blog/long-distance-relationship-anniversary-ideas
   - https://lovefarfox.com/blog/long-distance-relationship-nicknames
   - https://lovefarfox.com/quiz
4. (Optional) Also add the site to **Bing Webmaster Tools** — it can import directly
   from Google Search Console in one click, and Bing indexing is already being fed by
   the IndexNow pings we set up.

That's the entire Google indexing task. After this, indexing is days, not weeks.

## Part 2 — Backlinks (the #1 ranking factor)

Ranking is mostly about other sites linking to you. The statistics post is built to
earn links; the rest is outreach. Work top-down — the first items are highest ROI.

### 2a. App directories (do all — easy, permanent backlinks)

Submit Far Fox to each. Most take <10 min. Use the blurb below.

- Product Hunt — https://www.producthunt.com/posts/new (see launch copy in 2b)
- AlternativeTo — https://alternativeto.net (list as alternative to "Couple", "Between", "Paired")
- There's An AI For That / app aggregators
- SaaSHub, AppAdvice, app listing sites for iOS/Android
- Capterra / G2 (if you want B2C-ish presence)
- Relationship/couples app roundup sites (search "best apps for long distance couples" — email each author, see 2d)

**Standard directory blurb (copy-paste):**
> **Far Fox** is a free couples app for long-distance relationships. Instead of being
> another messaging app, it's built around daily rituals — daily questions, love letters
> on themed stationery, shared photo timelines, voice notes, couple quizzes, countdowns,
> and a kawaii fox companion that grows the more you show up for each other. Free on iOS,
> Android, and web. https://lovefarfox.com

### 2c. Reddit (genuine value, NOT spam — links in profile + occasional contextual share)

Put https://lovefarfox.com in your Reddit profile. Then be genuinely helpful in:
- r/LongDistance (500k+), r/LDR, r/MilitarySO, r/relationship_advice

**Rule:** never drop a raw link as a top-level promo. Instead, answer questions
thoughtfully and reference a guide when it truly helps. Example reply that's allowed
to link:
> This is such a common wall around month 3 — it's almost universal. We actually mapped
> the whole emotional arc month-by-month here if it helps: [link to the timeline post].
> The short version: it's not you, it's communication fatigue, and it's fixable.

High-value posts you can make (value-first, link in comments if asked):
- "I rounded up the actual long-distance relationship statistics with sources" → links naturally to the stats post
- "47 long-distance date ideas that aren't just 'watch a movie on call'" → date-ideas post
